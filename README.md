# 🎨 Paleta de Colores – Marca

## 🟤 Paleta principal

### 🟤 Marrón oscuro (principal)
- **Uso:** textos, títulos  
- **HEX:** `#3B2416`  
- **Sensación:** confianza, tradición, calidad  

### 🟡 Dorado / Mostaza (acento)
- **Uso:** líneas, íconos, detalles  
- **HEX:** `#C89B3C`  
- **Sensación:** premium, valor, elegancia  

### 🟤 Marrón medio
- **Uso:** elementos secundarios, fondos suaves  
- **HEX:** `#6B4A2F`  

### 🤍 Beige claro (fondo)
- **Uso:** fondo general  
- **HEX:** `#F5EBDD`  
- **Sensación:** limpio, gastronómico, cálido  


## 🎨 Colores secundarios (del logo)

### 🟢 Verde oliva
- **Uso:** detalles naturales (productos, frescura)  
- **HEX:** `#6E7F3F`  

### 🔵 Azul montaña
- **Uso:** solo gráfico / logo (**no usar en textos**)  
- **HEX:** `#4A6C8C`  

### 🥛 Blanco crema
- **Uso:** contraste / fondos  
- **HEX:** `#FFF8EE`  


## 🧠 Cómo usar la paleta

Para mantener una estética profesional y consistente:

- **Fondo:** beige claro  
- **Títulos:** marrón oscuro  
- **Destacados / promociones:** dorado  
- **Líneas y separadores:** marrón medio  
- **WhatsApp:** mantener su verde oficial  


## 🔥 Tip Pro

Evitar el uso de colores que rompan la identidad visual:

- ❌ Rojo  
- ❌ Violeta  
- ❌ Colores saturados o fuera de la paleta  

👉 Mantener coherencia cromática refuerza la percepción de marca.
